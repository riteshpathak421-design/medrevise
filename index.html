⏰ Set up Siri reminder
How it works: iPhone's Shortcuts app runs a daily automation at any time you set — full sound, lock screen, works offline. No extra apps needed!
1
Open the Shortcuts app
Already on your iPhone. Search Shortcuts in Spotlight if you can't find it.
2
Tap "Automation" → "New Automation"
Tap the Automation tab at the bottom, then the + button at top right.
3
Choose "Time of Day"
Set your time e.g. 8:00 AM → Repeat: Daily → tap Next.
4
Add action → "Show Notification"
Tap Add Action → search notification → pick Show Notification. Set text to: Time to revise! Open MedRevise 🌸
5
Turn OFF "Ask Before Running"
Tap Next → toggle Ask Before Running to OFF → tap Done.
6
Done! 🎉
Every day at your time the iPhone rings. Tap notification → open MedRevise → see what's due!
